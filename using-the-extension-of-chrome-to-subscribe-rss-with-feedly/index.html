如何在 Chrome 中方便地添加 RSS 订阅源至 Feedly

2015-08-09 Category: 随笔 Chrome RSS 分享 loading views
  1. 1.安装 Chrome 扩展程序 RSS Subscription Extension
  2. 2.配置扩展使用的 RSS 阅读器
  3. 3.关于扩展的使用
窝也经常在网上到处乱逛，从别人的友链出发跳来跳去，有时候找到了一些内容质量高的博客就 RSS 订阅至 feedly ，但是每一次都要打开 feedly 再添加，确实是不便。Google 了一下找到了解决方案，整理如下：

1.安装 Chrome 扩展程序 RSS Subscription Extension

2.配置扩展使用的 RSS 阅读器

默认的 RSS 阅读器是 Bloglines 和 My Yahoo，然而 Bloglines 老早就停止服务了，My Yahoo 窝也没用过，窝自然还是要继续使用窝大 Feedly 辣~下面的内容参考了这篇文章。

进入 RSS Subscription Extension 配置页 -> 添加 -> 填入如下网址

https://www.feedly.com/home#subscription/feed/%s

保存后即可一键订阅 RSS 至 feedly 。

3.关于扩展的使用
